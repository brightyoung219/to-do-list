# React To Do List

바로가기 : https://brightyoung219.github.io/to-do-list/


### 프로젝트 목표
- React를 사용하여 간단한 to do list 기능 구현

### 프로젝트에 사용한 기술
- useState 리액트 훅
- map() 함수를 이용한 배열 렌더링

### 프로젝트 기능 설명
- input에 내용 입력 후 엔터키 혹은 버튼 클릭 시 하단에 할 일 목록이 저장됩니다.
- 제목 우측에 작성된 리스트의 개수가 표시됩니다.
